Live Event Chat

Drive customer engagement during high-traffic events by enabling real-time community conversations. Consumers can interact with each other and voice opinions during high-profile events, TV programs, webcasts, and more.

Live Event Chat differs from Commenting in that each instance of Chat must be provisioned in the dashboard, but the instances can easily be used in many locations.

Try out these features on this example implementation:
  • After signing into site, you are also signed into Live Event Chat
  • Like a post
  • Comment on a comment
  • Share comment to Facebook and Twitter (if logged in with that IDP)
  • Dirty word filter prevents offensive posts
  • Comments added on this page also show in Site Activity Feed
  • Comment sort order choices
  • Users can delete their own posts

Note that this demo is based on the Backplane 1.2 specification. For more details, see:
http://backplanex.com/specification-documents/backplane-specification-1-2/
