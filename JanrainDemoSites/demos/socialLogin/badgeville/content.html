Demo of Janrain Social Login with Badgeville.

Try viewing source in the page to see what's going on here.

There is one BV mission set up: "Getting Started". To complete it, you need to get 3 badges:
1) perform an initial social login
2) with the same account, click the link which appears below after signing in. (This simulates performing some site-specific activity like editing a profile page).
3) refresh the page and then perform a return visit with the same account.

(note that the default Badgeville messaging may also have links -- "complete my profile",
"search for restaurants", etc. How these appear depends on how you set up your own instance of Badgeville,
and are not specific to the functionality of this demo integration.)
