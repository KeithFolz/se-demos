This is a simple demo of Janrain Enterprise with Badgeville:

There is one BV mission set up: "Getting Started". To complete it, you need to get 3 badges:
1) perform an initial registration (using a social provider - this demo is only showing social integration at this time)
2) sign out and then perform a return visit with the same account
3) with the same account, edit your Janrain Enterprise profile (the "edit profile" link which will appear above) once.

(note that the default Badgeville messaging will also have links -- "complete my profile",
"search for restaurants", etc. How these appear depends on how you set up your own instance of Badgeville,
and are not specific to the functionality of this demo integration.)
