Simple demo of Enterprise with Janrain SSO.

Setting up Janrain SSO requires a few core steps:

  1. Add SSO settings.
  2. Include a Cross Domain Receiver page.
  3. Implement additional SSO event hooks (see janrain-init.js)
  4. Implement a logout page (see: this page).

Settings and examples for all of these will be provided to you by your Janrain Tech Lead during deployment.

In order to test SSO here, you can try this page, which sits on a different domain but points to the same Janrain Enterprise instance and has the same SSO configuration. You'll see that signing in or signing out from either page on either domain will have the parallel effect on the other. Be sure to view the Javascript console for more helpful information.
